yarn创建vue项目

发表于 2022-07-11 更新于 2022-07-12

安装yarn:
npm install -g yarn
检查安装成功:yarn -v
https://blog.csdn.net/Kirito12/article/details/119655425

1.

yarn global add @vue/cli //全局安装vue脚手架
yarn global bin //查找yarn安装目录

添加到系统变量得path里面

2、

vue create 项目名 //创建项目目录

（1）执行完这段命令后会有选项，根据自己的需求进行选择，一般选最后一个
（2）进入下一个选项，选择要安装的插件依赖，根据自己的需求来进行选择，按空格键就可以进行选择
（3）因为上一步我选择了css Pre-processors，所以下一步会让我选择css的预处理语言
（4）下一个选项一般选package.json
(5)此选项默认不保存就行，直接回车下一步
（5)环境搭建成功，cd进入项目目录
（6）yarn serve启动项目就可以了

创建uniapp (上接第一条)

  1. vue create -p dcloudio/uni-preset-vue 项目名
    (安装失败：npm uninstall vue-cli -g 或 yarn global remove vue-cli （@vue/cli） 卸载)
    （官方文档：https://uniapp.dcloud.io/quickstart-cli.html）

git常用命令步骤

发表于 2022-07-01 更新于 2022-07-12

1.克隆仓库

git clone 仓库地址

2.创建自己的分支

git branch 分支名字

查看当前分支

git branch -v

查看已有分支

git branch

删除分支

git branch -d 分支名字

3.将文件添加到缓存(.代表全部添加)

git add .

4.提交到版本库

git commit -m "提交的描述信息"

5.将本地内容推送到远程仓库

git push origin 分支名字

6.切换分支

git checkout 分支名字

7.合并分支

git merge 分支名字

8.拉取远程仓库代码

git pull origin 分支名字
